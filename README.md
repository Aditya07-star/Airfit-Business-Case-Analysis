## Airfit Business Case Analysis

**Tools:** Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

**About:**
Analyzed treadmill purchase data to segment customers 
into 3 profiles based on age, income, fitness level and usage patterns.

**Dataset:**
https://d2beiqkhq929f0.cloudfront.net/public_assets/assets/000/001/125/original/aerofit_treadmill.csv?1639992749

**Key Findings:**
- KP281 attracts budget-conscious beginners (age 20s-30s)
- KP481 suits regular working professionals (mid 30s)
- KP781 is chosen by high-income serious runners (late 30s-40s)

**Business Recommendations:**
- KP281 → Target students and beginners with EMI options
- KP481 → Target working professionals with bundle deals
- KP781 → Target high-income buyers with premium marketing
